# MinuteFlow Studio

PDF 회의록을 현장에서 수정하고 서명해 최종 PDF로 저장하는 로컬 워크스페이스다.

## 핵심 흐름

1. PDF 열기
2. 필요 시 텍스트 수정/삭제
3. 서명(직접 입력 또는 서명 파일)
4. PDF 다운로드
5. 문서/서명 프리셋 저장

## 포함 기능

- PDF 업로드 후 페이지 미리보기
- 기존 텍스트 선택 후 덮어쓰기 수정/삭제
- 텍스트 추가, 가림판, 사진 삽입
- 마우스/터치/펜 현장 서명
- 투명 PNG/JPG/WebP 서명 파일 삽입
- 문서 프리셋/서명 프리셋 `IndexedDB` 로컬 저장
- 최종 PDF 다운로드

## 실행 방법

1. 정적 파일 서버로 프로젝트를 연다.
2. `index.html`에 접속한다.
3. PDF를 열고 수정/서명 후 저장한다.

## 구조

- `index.html`: 랜딩 + 작업 UI
- `styles.css`: 반응형 스타일
- `app.js`: PDF 로드, 편집, 서명, 프리셋, 저장 로직

## 제한

- 현재 저장 포맷은 PDF만 지원한다.
- 서버 저장, 결재선, 권한, 이력 관리 기능은 없다.
